# Smart Health Tracker

## Overview

Smart Health Tracker is a full-stack MERN application designed to help users monitor and manage their daily health metrics through an intuitive dashboard. The platform enables users to record vital health information, analyze weekly trends, generate health reports, and maintain a personalized health profile.

The application provides a centralized health monitoring system with secure user authentication, interactive visualizations, and automated report generation to help users better understand their health patterns.

### Frontend Setup

```bash
cd frontend
npm install
npm start
```

Frontend will run on:

```text
http://localhost:3000
```

### Backend Setup

```bash
cd backend
npm install
node server.js
```

Backend will run on:

```text
http://localhost:5000
```

### MongoDB

Ensure MongoDB is installed and running locally before starting the backend server.

## Application Workflow

1. User creates an account or logs in.
2. User enters daily health information.
3. Health metrics are stored securely in MongoDB.
4. Dashboard visualizes user data through charts and summaries.
5. Weekly analytics provide health insights.
6. Users can download health reports for record keeping.
